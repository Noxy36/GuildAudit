Terms of Service

GuildAudit provides analytics and engagement insights for Discord servers.

By installing and using GuildAudit, server administrators agree that the bot may analyze publicly accessible messages and channel activity in order to generate engagement metrics.

GuildAudit does not permanently store message content.

The service is provided "as is" without guarantees.

Privacy Policy

GuildAudit processes limited Discord data in order to generate analytics reports.

This may include:

  • User IDs
  • Channel IDs
  • Message metadata
  • Aggregated activity metrics

GuildAudit does not sell or distribute personal data.

All data is used only to generate reports for the server where the bot is installed.
